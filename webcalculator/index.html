Web Calculator

This is a web calculator written in pure HTML/CSS/Javascript without using any external libraries or frameworks and is designed to work well with screeen readers.

Calculations can be entered via the buttons on screen or via the keyboard using the number keys, period, and the following keys for operators:

Use i for multiplicative inverse (1 divided by x)

Use s for square

Use r for square root

Use n for negate (0 minus x)

Use / for divide

Use * for multiply

Use + for plus

Use − for minus

Use = for equals
